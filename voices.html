How to use this page

Each photo has an audio-description button next to it that plays a short spoken description on request — nothing plays automatically. Videos use an accessible player with a caption toggle and an audio-description toggle in the control bar. The "Sounds of the bay" button in the header turns on an optional ambient soundscape; it is off by default and pauses automatically whenever an audio description is playing. Every part of this page can be reached and operated using only a keyboard.
